# 彩球连珠 · Color Lines（网页复刻版）

经典 Lines 98 / 彩球连珠的网页复刻，**零依赖、零构建**，双击 `index.html` 即玩。
内置**完整对局记录**与**单步复盘**，方便事后回看每一手。

## 玩法

- N×N 棋盘（默认 9×9），随机出现 7 种颜色的球。
- 点一个球选中，再点一个**空格**移动；只能沿**联通的空格路径**走，被挡住则无法移动。
- 同色连成一线（横/竖/斜）达到设定长度即消除得分。
- 每走一步且未消除时，棋盘随机生成新球（带"下一批"预告）。
- 棋盘填满即结束。

### 可调设置

| 设置 | 选项 | 默认 |
| --- | --- | --- |
| 连线长度 | 4 / 5 | 5（经典）；选 4 即你描述的版本 |
| 棋盘 | 7 / 9 / 11 | 9 |
| 颜色数 | 5 / 6 / 7 | 7 |
| 每步生成 | 2 / 3 / 4 | 3 |

## 复盘（核心功能）

每一局都被记成「初始局面 + 带时间戳的事件流」（移动 / 生成 / 消除 / 结束），
存在浏览器 `localStorage`。切到「复盘」标签：

- 左栏选历史对局；
- 中间棋盘可**单步前进/后退、拖动进度条、自动播放**；
- 右栏是**事件流**，点任意一条直接跳到那一手。

数据可**导出 JSON**（单局或全部），用于离线分析。

## 运行

直接打开 `index.html` 即可。若浏览器对本地文件有限制，可起个静态服务器：

```bash
# 任选其一
python -m http.server 8000
npx serve .
```

## 代码结构

```
js/
  rng.js       种子随机数
  game.js      纯逻辑：棋盘 / BFS 寻路 / 连线判定 / 生成
  recorder.js  对局记录 + localStorage 持久化 + 导出
  replay.js    事件流 → 帧序列 的复盘重建
  ui.js        棋盘渲染助手
  main.js      游戏与复盘的交互编排
```

逻辑层（game.js）无 DOM、无副作用；复盘完全由事件流重建，不依赖随机数，旧录像永远能放。
