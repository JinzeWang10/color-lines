# 彩球连珠 · Color Lines（网页复刻版）

经典 Lines 98 / 彩球连珠的网页复刻，**零依赖、零构建**，双击 `index.html` 即玩。
为「送给爸爸的父亲节礼物」打造：补齐了和单机版的体验差距（音效、撤销、最高分、结算画面），
并内置单机版没有的**完整对局记录 + 单步复盘 + 数据看板**，让喜欢这游戏的人能回看、能变强。

## 玩法

- N×N 棋盘（默认 9×9），随机出现 7 种颜色的球。
- 点一个球选中，再点一个**空格**移动；只能沿**联通的空格路径**走，被挡住则无法移动。
- 同色连成一线（横/竖/斜）达到设定长度即消除得分。
- 每走一步且未消除时，棋盘随机生成新球（带"下一批"预告）。
- 棋盘填满即结束，弹出结算（本局数据 + 是否破纪录 + 鼓励语）。

**默认 5 连消除**（经典 Lines 98），设置里可切到 4 连。

### 体验功能

- 🔊 **音效**：选球 / 移动 / 消除 / 走不通 / 通关 / 破纪录，全用 WebAudio 实时合成，无音频文件。可一键静音。
- ↶ **撤销上一步**：手滑点错不怕（`Ctrl+Z` 或 `U` 键也行）。完整还原棋盘、分数与随机状态。
- 🏆 **最高分**：本机持久保存，结算时若破纪录有专属提示音和徽章。
- 适老化：更大的球与按钮、清晰的选中/走不通反馈。

### 防沉迷

每天累计**真正在玩**的时间达 **1.5 小时**即锁定，弹出"明天再来"遮罩，过本地零点自动解除。

- 只在「游戏」页 + 页面可见 + 最近 2 分钟有操作时才计时；看复盘、挂机、切走都不计。
- 剩 10 分钟时温柔提醒一次，不突然掐断。
- 上限是代码常量，**界面里改不了**（否则形同虚设）。要调时间，改 `js/limit.js` 顶部的 `dailyLimitSec` 再 `node build.js` 重新打包。
- 这是**软限制**：清除浏览器数据或改系统时钟可绕过，对长辈足够；真正绕不过需操作系统级管控。

### 可调设置

| 设置 | 选项 | 默认 |
| --- | --- | --- |
| 连线长度 | 4 / 5 | **5** |
| 棋盘 | 7 / 9 / 11 | 9 |
| 颜色数 | 5 / 6 / 7 | 7 |
| 每步生成 | 2 / 3 / 4 | 3 |

## 复盘（核心功能）

每一局都被记成「初始局面 + 带时间戳的事件流」（移动 / 生成 / 消除 / 结束），
存在浏览器 `localStorage`。切到「复盘」标签：

- 顶部**历史看板**：最高分、平均分、总局数、得分趋势小图。
- 选某局看**六项指标**：得分 / 步数 / 消除球数 / 最大连消 / 每步均消 / 用时。
- 中间棋盘可**单步前进/后退（← →）、拖进度条、自动播放**。
- 右栏**事件流**，点任意一条直接跳到那一手。

数据可**导出 JSON**（单局或全部），用于离线分析。

## 送给爸爸：打包成一个文件

```bash
node build.js
```

会在 `dist/color-lines.html` 生成一个**完全独立的单文件**（CSS/JS 全内联，约 54KB）。
直接微信/邮件发给他，双击就能玩，不用管文件夹和相对路径。

## 运行（开发时）

直接打开 `index.html` 即可。若浏览器对本地多文件有限制，可起个静态服务器：

```bash
python -m http.server 8000   # 或 npx serve .
```


## 代码结构

```
js/
  rng.js       种子随机数（含 get/setState，供撤销回滚）
  limit.js     防沉迷：按自然日累计活跃时间，达上限锁定
  sound.js     WebAudio 合成音效 + 静音
  game.js      纯逻辑：棋盘 / BFS 寻路 / 连线判定 / 生成 / 快照还原
  recorder.js  对局记录 + localStorage 持久化 + 最高分 + 导出
  replay.js    事件流 → 帧序列 的复盘重建
  stats.js     单局指标 / 历史聚合 / 折线图
  ui.js        棋盘渲染助手
  main.js      游戏与复盘的交互编排
build.js       打包单文件
test/          逻辑自测
```

逻辑层（game.js）无 DOM、无副作用；复盘完全由事件流重建、不依赖随机数，旧录像永远能放。
